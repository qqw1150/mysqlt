首页 开始使用MySQL

摘要：在本教程中，您将学习如何使用mysql命令行客户端和MySQL Workbench连接到MySQL Server。

一旦安装了MySQL服务器，就可以使用任何客户端程序（例如mysql命令行客户端和MySQL工作台）连接到它。

使用mysql命令行客户端连接到MySQL

mysql是一个命令行客户端程序，允许您以交互和非交互模式与MySQL进行交互。

mysql命令行客户端通常位于MySQL安装文件夹的bin目录中。

要调用mysql程序，只需导航到MySQL安装文件夹的bin目录并键入：

1
mysql

如果mysql程序已经在中PATH，则可以使用mysql命令简单地调用它。

要连接到MySQL服务器，请使用以下命令：

1
shell>mysql -u root -p

-u root表示您使用用户帐户连接到MySQL Server root。

-p 指示mysql提示输入密码。

您输入用户帐户root的密码，然后按Enter：

1
Enter password: ********

如果一切正常，您将使用以下命令连接到MySQL服务器：

1
mysql>

要显示当前服务器中的数据库，请使用以下SHOW DATABASES语句：

1
mysql> show databases;

使用MySQL Workbench连接到MySQL

步骤1.启动MySQL Workbench。

您可以使用“数据库”>“连接到数据库...”菜单连接到MySQL服务器，或单击“ MySQL连接”旁边的+按钮。

只需单击“ MySQL连接”旁边的+按钮即可继续。

步骤2.输入连接名称，例如Localhost。您可以随意命名。默认情况下，用户名是root。如果使用其他用户帐户，则可以在“用户名”文本框中进行更改。

步骤3.单击Store in Vault ...按钮输入提供的用户帐户的密码。将显示一个窗口。您输入密码，然后单击确定按钮。

步骤4.单击“测试连接”按钮以测试与MySQL Server的连接是否成功。如果成功建立连接，然后单击“确定”按钮。

步骤5.单击确定按钮以保存连接。
